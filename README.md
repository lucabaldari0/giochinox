# Shell vs Trash v1.0

A 2D arcade game built entirely in Python with Pygame.

You play as a sea turtle fighting ocean pollution — dodge and shoot through waves of trash across 3 stages, then survive endless mode.

## Controls

- **Arrow keys** or **WASD** — move the turtle
- Shooting is **automatic**
- **ESC** — pause

## Features

- 3 stages with increasing difficulty + infinite survival mode
- 3 bosses, each with unique attack patterns
- Temporary and permanent power-ups
- Animated sprites

## Installation

```bash
git clone https://github.com/lucabaldari0/shell-vs-trash.git
cd giochinox
pip install -r requirements.txt
python main.py
```

## Requirements

- Python 3.13+
- pygame 2.6.1
